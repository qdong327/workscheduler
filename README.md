# Work Day Scheduler

## Description:
A simple work scheduler that allows users to add, remove and modfy events to each hour. Bonus features include ticking time clock as well as countdown to next hourly appointment.


## Technologies Used:
* Structure:
  - HTML
* Design/Frameworks:
  - CSS
  - Bootstrap
* Functionality:
  - Javascript
  - jQuery
  - Moment.js
